# AI Prompt — Personal Portfolio Website

Goal: Build a modern, personal portfolio website scaffold to showcase projects and skills, and contact details. Deliver a production-ready, deployable codebase. 
Requirements:
- Audience: potential employers, collaborators, and clients.
- Style: clean and minimal, with emphasis on readability.
- Responsiveness: mobile-first, works on phones, tablets, and desktops.

Tech & deliverables:
- Primary stack: Next.js with TypeScript + Tailwind CSS. 
- Deliverables: full source repo, production deployment instructions (Through Github Pages).

Design
- Create a theme color around #937055. Focus on minimalism with small text size and structures.
- Follow a similar design to https://www.craftz.dog (source code at https://www.craftz.dog) with wider horizontal margins.
- Header with tabs for each page
- Overview page from top to bottom: Hero with a placeholder image for pfp, Contact, Short Introduction, Work Experience, Skills Footer with logos of languages. Do not include any skills without badges on image shield. 
    - Placeholder image on the left.
    - Vertical timeline work experience.
    - Do not use a card for the Hero. 
- Projects page: Placeholder cards for linking to github projects.
- Dashboard: Placeholder page.

Acceptance criteria:
- Fully responsive across common breakpoints.
- Passes basic accessibility checks.
- SEO metadata present for main pages.
- Clean, readable code with TypeScript types.
